Problem Statements On Insertion Sort

You have an array of integers, and you need to sort it in ascending order using the Insertion Sort algorithm. Implement a solution to achieve this.

Solution :
public static void insertionSort(int[] arr) {
    int n = arr.length;
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;

        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}
